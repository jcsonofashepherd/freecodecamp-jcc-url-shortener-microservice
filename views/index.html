Free Code Camp URL Shortener Microservice

Part 3 of Free Code Camp Backend Challenges

Try it out:
type in /new/[url to be shortened] in the url

e.g. input:
https://jcasabar-url-shortener.glitch.me/new/https://www.google.com

e.g. output (shortened url may vary):
{"original_url":"https://www.google.com","short_url":"https://jcasabar-test.glitch.me/7764"}
